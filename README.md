## 📦 Setup Instructions

### 1. Clone the repo
```bash
git clone https://github.com/DeepKnowledge1/VisionX-AI.git
cd VisionX-AI
````

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

### 5. Deploy to GitHub Pages

```bash
npm run deploy
```
